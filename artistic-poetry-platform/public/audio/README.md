# Audio Files Directory

Place your background music file here.

## Instructions:

1. Name your background music file: `background-music.mp3`
2. Place it in this directory: `public/audio/background-music.mp3`
3. The app will automatically use it

## File Requirements:
- Format: MP3
- Size: Under 5MB recommended
- Bitrate: 128kbps recommended
- Type: Instrumental/ambient (no lyrics)

## Current Setup:
The app looks for: `/audio/background-music.mp3`

If the file doesn't exist, no music will play (no errors).
